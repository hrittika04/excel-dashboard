# 📊 Sales Performance Dashboard

## 📌 Overview

This project contains an interactive **Sales Performance Dashboard** built in Microsoft Excel.
It provides insights into sales executives' performance across different regions by tracking daily sales, total sales, and target achievement.

The dashboard helps identify:

* Top-performing sales executives
* Regional performance trends
* Target achievement and gaps

---

## 🚀 Key Features

* 📈 Daily sales tracking (Day 1 to Day 5)
* 👨‍💼 Sales executive-wise performance analysis
* 🌍 Region-wise comparison
* 🎯 Target vs Actual sales tracking
* 📊 Target achievement (%) and shortfall analysis
* 📉 Automated calculations for performance metrics

---

## 🧾 Dataset Description

The dataset includes the following fields:

* **Emp Code** – Unique identifier for each sales executive
* **Sales Executive** – Name of the employee
* **Region** – Assigned sales region
* **Day1–Day5** – Daily sales values
* **Total Sales** – Sum of all daily sales
* **Target** – Assigned sales target
* **Target Hit %** – Percentage of target achieved
* **Away From Target %** – Remaining gap to reach target

---

## ⚙️ How to Use

1. Download the Excel file from this repository
2. Open using Microsoft Excel (2016 or later recommended)
3. Enable macros if prompted (`.xlsm` file)
4. Navigate through the dashboard sheet
5. Use filters/slicers (if available) to explore performance
